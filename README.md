### 영자 신문을 앱으로 간편하게, 해외뉴우스

해외 뉴스로 세상을 바라보는 시야를 넓히고, 영어 공부까지 !

- 현재 해외에서 인기 급상승중인 주제와 카테고리별 기사들의 목록을 볼 수 있어요.
- 검색을 통해 원하는 주제의 기사를 찾을 수 있어요.
- 마음에 드는 기사를 공유해보세요 !
